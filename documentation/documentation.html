File Structure

- z-theme
  - assets
    - css (Template and Plugins CSS)

    - images (Template Images)

    - js (Template and Plugins JS)
        - page (JS to run in specific pages.)

- skin (Template Predifined Colors CSS and Less file to change the theme color with just one line of code)

- documentation (Template Documentation)

CSS Structure

- theme.css (includes styling of all elements and sections)

- photographer.css (styling of photographer page)

- coming-soon-404.css (styling of coming-soon and 404 pages)

- carousel-animate.css (styling of intro carousel preloading and animation)

- magnific-popup.css (magnific-popup default styling)

- owl-carousel.css (owl carousel default styling)

- owl-theme-animate.css (owl carousel default styling)

- prettify.css (styling of google prettify code highlighting)

Customization & Styles

Colors

There are 20 predefined color files in "skin" folder, just select one, rename it to "theme.css then replace it with your existing theme.css file in the assets > css folder.

Manual color with less

You can change the theme color with just 1 line of code by changing the @primarycolor in color.less file in skins folder.

Layout (Normal/Boxed)

Template has two layout mode, default one is wide. You can use the Boxed version by adding container boxed to body. Example

<body class="container boxed">
